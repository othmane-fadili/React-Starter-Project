for creating the react project starter use this command:
->npx create-react-app myportfolio or npm init react-app myportfolio
if you get an error you can solve it with:
->npm uninstall -g create-react-app
->npx clear-npx-cache
->npx create-react-app myportfolio

during the creation of this Project i used this versions:
->node -v: v14.15.5
->npm -v: 8.19.2
->npx -v: 8.19.2

for runing this project use:
->cd frontend project name
->npm start
